Tommy asked us to help them create Rich Media Ads for Netflix using GeoLocation, MultiVideo & Swipe – in other words, very complex and innovative HTML5 ads.

Sense8 and House of Cards

We helped create the campaigns for both shows. For Sense8, the campaign was quite extense, including TV interface backgrounds, HTML5 ads with interactive images, and with dynamic videos.

For House of Cards, the challenge was the geo-targeting, which we created versions for different cities in England, based on the best quotes for each region.

SENSE8

Multi-video
Dynamic Ads

This one was a lot of fun. For the new show Sense8, they wanted to present all characters, the trailer and an intro video.

For a bit more wow, I created an edit of the main trailer in After Effects, with explosions and all, and that would play before the branded netflix video.

About this Image

Preview Ad

User Journey

The plan then was to have a static version as a poster, and when interacted, it would jump seamlessly into the interactive and higher quality version. I ended up using SVG images so it would scale nicely on retina screens, but the poster image was a very compressed jpeg.

Technical limitations
For the change of assets to be really smooth, the initial load had already the libraries download thus the file size challenge. Instead of waiting for the image swap and the extra load, there was a quick fade, and the assets would animate and be ready for swipe.

The user would then drag, tap or click the arrows and swipe to match the characters blocks and get to a related end frame and exit URL.

HOUSE OF CARDS

Geo-targeted Dynamic Ads

The ad had the recent dynamic components in Doubleclick. The geolocation was a challenge as it needed detailed instructions, and the use of Rules to test.

Essentially, the copy would say the City name, and change the show assets, which had a Gif animation and images, all dynamic from DoubleClick Assets.

House Of Cards Dynamic Ad

Functionality and Testing
